YouTube Browser Bookmark Tagger

Description

This extension activates when you open a YouTube video.

If the video exists in your browser bookmarks, then buttons will be displayed below the YouTube video player. (as shown above)

When one of these buttons are clicked, the #tag will be toggled on/off in your browser bookmark. (adds and removes the tag from the browser bookmark title)

To change the tags available, edit the settings below.

Settings

Tag must not contain whitespace or # characters

Each sub-array is displayed as a seperate row of tags
